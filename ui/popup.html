InterceptX — Usage Guide

1. Introduction

InterceptX allows you to dynamically modify request/response headers, redirect URLs, and filter modifications by tab domain or request URL patterns.

2. Usage Modes

  • Popup Mode: Click the extension icon in the toolbar for quick additions.
  • Dashboard Mode: Click the open icon next to the profile name to open a dual-column workspace in a full tab.

3. Rule Modifications

  • Headers: Add header injection rules for Requests (Set, Append, Remove) and Responses (Set, Append, Remove).
  • Redirects: Match incoming request URLs using regular expressions (RE2 syntax) and redirect them to target URLs (supports capture groups like \1).

4. Filter Types

  • Match: Apply rule modifications only if the request URL matches this wildcard pattern.
  • Exclude: Bypass rule modifications if the request URL matches this pattern (uses high-priority allow action).
  • Domain: Restrict rules to requests initiated from this tab domain.

5. Import & Export

Save and restore configurations instantly as JSON files using the Export and Import buttons in the header toolbar.
